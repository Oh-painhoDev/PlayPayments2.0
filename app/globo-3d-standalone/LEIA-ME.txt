🚀 COMO COMEÇAR (3 PASSOS)
═══════════════════════════════════════════════════════════════════════

PASSO 1: Execute o script de download
─────────────────────────────────────

Windows:
    Duplo clique em: download-deps.bat

Mac/Linux:
    chmod +x download-deps.sh
    ./download-deps.sh

   ⏱️  Espera: ~1 minuto para baixar as bibliotecas

PASSO 2: Inicie um servidor
─────────────────────────────

   PHP (seu Laragon):
   $ php -S localhost:8000

   Python:
   $ python -m http.server 8000

   Node.js:
   $ npx serve

PASSO 3: Abra no navegador
──────────────────────────

   http://localhost:8000

   🎉 Pronto! O globo deve aparecer girando!

🎨 CUSTOMIZAÇÃO RÁPIDA
═════════════════════════════════════════════════════════════════════

Adicionar mais países:
  → Edite: js/app.js
  → Procure: const salesData = [...]
  → Adicione novos objetos com { lat, lng, value, country, sales }

Mudar cores:
  → Edite: js/app.js
  → Procure: .pointColor(d => {...})
  → Troque os valores hex

Mudar velocidade de rotação:
  → Edite: js/app.js
  → Procure: autoRotateSpeed = 2;
  → Aumente ou diminua o valor
